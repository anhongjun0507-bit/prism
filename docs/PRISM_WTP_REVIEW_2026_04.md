# PRISM 지불 의사(WTP) 평가 — 2026-04-21

## Executive Summary

**판정: 조건부 Go (Payment Gate 오픈 자체는 OK, 다만 현재 구조로는 월 매출 1억 도달 불가능)**

- **₩9,900 낼 만한 앱인가?** — **조건부 Yes.** 제품 자체의 기술·UI는 "결제 받을 수 있는 최저선"은 넘겼다. 하지만 "유저가 자발적으로 지갑을 여는" 수준은 아니다.
- **5명 페르소나 중 "바로 낸다"** — **1명** (지민/에세이 첨삭). "생각해본다" 1명(민준), "무료로 더 써본다" 2명(서연·재우), "안 낸다" 1명(혜진 어머니).
- **가장 큰 지불 장벽** — **"5분 안에 ChatGPT 대비 차별점을 몸으로 느끼게 하는 순간"이 없음.** 프로필 주입 RAG가 코드에는 있지만, UI에서 "아, 이건 ChatGPT가 못 주는 답이다"가 **시각적으로 드러나는 장치**가 부재.
- **3개월 후 가중 평균 월 매출 예측** — **₩1,800,000~2,400,000** (월 180~240만 원). 낙관 시나리오에서도 월 600만 원 수준. 1억은 제품/획득 전략 재설계 없이는 불가.
- **출시 권고** — **Go, 단 "결제 gate 오픈 + 유저 행동 데이터 수집 2주 → 측정 후 개선" 전제.** Pivot 없이 유료화 지표만 계측하는 MVP 단계로 진입.

## Phase A: 5명의 페르소나 — 구매 결정

### 페르소나 1: 민준 (외고 11학년, 이미 대치동 컨설팅 중)

**체험 요약:**
1. 가입 → 프로필 입력 (GPA 3.9, SAT 1480, TOEFL 105) → 대시보드.
2. /analysis에서 200개 대학 합격 확률 → Harvard "Reach 12%", UIUC CS "Target 62%" 같은 숫자 확인.
3. /chat에서 "내 스펙으로 Columbia EA 넣을 만한가?" 질문 → AI가 이름·GPA·SAT 실제로 반영한 답을 줌 + "분석 페이지로 이동" CTA 등장.

**결정: 생각 좀 해본다**

**핵심 이유:**
- 대치동 80만 원 컨설팅과 비교했을 때 **데이터 범위**(1,001개) 부분은 확실한 + 요소. 컨설턴트는 상위 30개만 커버함.
- 하지만 "컨설팅 비중을 줄일 만한 깊이"는 아직 느껴지지 않음. 컨설턴트는 "이번 달 너 에세이 2개 마감, 지금 UIUC 2번 essay supplement 먼저 써라" 같은 **주간 단위 맞춤 지시**를 줌. PRISM은 분석·조언 수준에서 멈춤.
- 민준은 "컨설팅 + PRISM 병행" 은 해볼 의향. 그러나 PRISM이 컨설팅 없앨 정도 → 현재 체험만으론 No.

**달라질 조건:**
- "매주 월요일 = 이번 주 할 일 3개를 **민준 너의 스케줄에 맞춰** 자동 제안" 기능이 있으면 바로 낸다.
- Premium 에세이 첨삭이 **실제 컨설턴트급 피드백 5페이지**를 뽑아주면 Premium 직행.

### 페르소나 2: 서연 (국제학교 10학년, 스펙 thin)

**체험 요약:**
1. 프로필 입력 (GPA 3.6, SAT 미응시, 전공 미정).
2. /analysis → "데이터가 부족해 분석이 부정확할 수 있어요" 라벨 + Reach만 가득. 좌절.
3. /chat에서 "뭐부터 해야 해?" → AI가 "SAT를 먼저 보고 전공을 1~2개로 좁혀봐" 같은 일반적인 답.

**결정: 무료로 더 써본다**

**핵심 이유:**
- 10학년은 **"첫 주에 뭘 알게 되는가"** 가 유일한 가치 축인데, 지금 PRISM은 "당신은 정보가 부족합니다"만 반복해서 말함. → 오히려 불안감 증폭.
- ChatGPT Free도 "SAT부터 봐라" 수준 답은 줌. 차별점 제로.
- 부모가 월 9,900원을 내주려면 **"이번 달 너는 이거 했다"** 는 가시적 진척이 필요. 지금은 스펙 입력만 한 상태.

**달라질 조건:**
- **"10학년 전용 12개월 로드맵"** + 매주 미션 1개 (ex: "이번 주: 너의 흥미 3가지 테스트 해보자") 형식. "AI 코치가 내 옆에 있다"는 감각을 만들면 Basic 바로 낸다.
- **스펙 분석이 없어도 쓸 수 있는 "탐색" 모드** (What-If 10학년 버전 = "만약 내가 SAT 1450 받으면 어디 갈 수 있어?") 가 무료에 있어야 유입 → 유료 전환 가능.

### 페르소나 3: 혜진 어머니 (민준 학부모, 기술 낮음)

**체험 요약:**
1. 아들이 보낸 링크 클릭 → 한국어 랜딩 → 가입 (아들 아이디 빌림).
2. 대시보드 → 영어 용어("Target/Reach/Safety") 섞여 있음. 학부모 리포트 메뉴 찾음.
3. /parent-report → Premium 전용 lock. "아들 분석 보려면 월 19,900원" 안내.

**결정: 안 낸다**

**핵심 이유:**
- 학부모가 "AI가 내 아들한테 뭘 해주는지" 를 **본인 화면에서** 확인하지 못함. 학부모 리포트가 Premium lock인데 **lock 해제 전 미리보기**가 없음 → "돈 내야 보여줌" 구조는 한국 학부모에게 치명적(샘플을 먼저 보여줘야 신뢰).
- Target/Reach/Safety 용어가 한국어 병기 없이 나옴 → "장난감" 이라는 의심 강화.
- 결정권자(학부모)에게 **체험 자산 0** = 컨설턴트 찾아가는 게 더 빠름.

**달라질 조건:**
- 학부모 리포트 **샘플 PDF 무료 공개** + "내 아들 이름 넣은 샘플 1회" 체험. "오, 이 정도 분석이 매달 오네" → 바로 결제.
- 대시보드에 **"학부모 공유 모드"** 토글 — 한국어만, 숫자보다 내러티브 위주. 아들이 켜주면 엄마가 본다.

### 페르소나 4: 재우 (자율고 12학년, 효율 중심)

**체험 요약:**
1. 10분 만에 가입·프로필·/analysis 훑기.
2. /chat "나 수능이랑 미국 병행하는데 이번 달 뭐 해?" → AI가 "Common App 에세이 먼저, SAT 한 번 더 볼지 결정" 같은 **구조적 답**.
3. suggest_actions CTA 3개 ("에세이 보기", "플래너 열기", "비교 모드") 가 붙음 → 실제로 /planner 이동.
4. /planner → 빈 화면. "직접 추가" 만 있음. 자동으로 to-do가 채워지지 않음.

**결정: 무료로 더 써본다**

**핵심 이유:**
- AI 답 품질은 좋음. CTA 버튼도 효율 중심 유저 취향에 맞음.
- 그러나 **플래너가 수동** → "10분 투자해서 이번 주 할 일" 을 AI가 **자동 세팅**하지 못함. 재우는 스스로 일정 짤 시간이 없는 유저.
- Free 5회/일 AI 상담만으로도 충분히 "감 잡기"는 가능 → 유료 전환 동기 약함.

**달라질 조건:**
- **"AI가 내 일정을 자동으로 플래너에 꽂아주는"** 기능. Chat → "다음 주 일정 자동 작성" 버튼 한 번에 /planner에 7개 이벤트 생성. 이거 되면 Basic 바로 낸다.
- 수능 디데이(D-215) 카운터 + 미국 마감 디데이(D-80) 카운터 대시보드 최상단 노출 → "이 앱이 나를 관리하고 있다" 감각.

### 페르소나 5: 지민 (12학년, 이미 Common App 작성 중)

**체험 요약:**
1. 가입 → 프로필 → /essays/review 로 직행 (오늘 쓰고 있는 Common App essay 붙여넣음).
2. AI 첨삭 → 10점 체계 점수 + 구체 개선안 ("3문단 두 번째 문장이 너무 추상적, 구체 사례 추가").
3. /chat에서 "Columbia supplement hook 아이디어 5개" 요청 → 지민 스펙 반영한 답.

**결정: 바로 낸다 (Premium ₩19,900)**

**핵심 이유:**
- **에세이 첨삭은 유일하게 "ChatGPT와 바로 비교 가능한 기능"**. ChatGPT Plus도 에세이 첨삭은 할 수 있지만 (a) 컨텍스트가 매번 날아감, (b) 점수 체계가 일관되지 않음, (c) 지원 대학 특성을 반영하지 못함.
- PRISM은 프로필·대학 데이터가 붙어있어서 **"Columbia가 좋아하는 톤" 반영 첨삭**이 일회성 GPT 프롬프트보다 일관적.
- 지민은 컨설턴트 바쁨 → 추가 첨삭 채널로 **월 ₩19,900은 적정**. 1개월 에세이 10번 첨삭 받으면 건당 ₩2,000. ROI 명백.

**달라질 조건:**
- 더 좋아지려면: **"Columbia 에세이 첨삭 전용 모드"** — 실제 합격자 에세이 비교 + 그 학교 전용 rubric. 그럼 Premium 연간 ₩149,000 바로 결제.

### 핵심 진단: **5분 안에 도달하는 가장 강한 aha moment는 #3 (AI 상담 맞춤 답변)**

**문제:** 이게 UI에서 **"평범한 챗"으로 보임**. ChatGPT와 시각적으로 구별 안 됨. 유저는 코드에서 서버 프로필 주입 + admission_results RAG가 일어나는 것을 **눈으로 볼 수 없음**.

### 강화 제안

**"AI가 참조한 근거"를 답변 하단에 보여주는 위젯**:
```
이 답변은 다음을 참고했어요:
✓ 민준님 프로필 (GPA 3.9, SAT 1480)
✓ 최근 Columbia CS 합격 사례 3건
✓ Common App Supplement 가이드
```
이 3줄만 렌더하면 **"이건 그냥 GPT가 아니구나"** 가 시각적으로 꽂힘. 개발 0.5일.

## Phase C: 경쟁 포지셔닝

### vs ChatGPT Plus (₩30,000/월)
- PRISM 월 ₩9,900은 ChatGPT Plus의 33% 가격. 가치로는 **"전문 영역만 25~40%"** 수준. 범용성에서 밀림.
- **대체재 아닌 보완재.** 대부분 유저는 둘 다 구독할 돈이 없음 → **택일**. 이미 ChatGPT Plus 쓰는 유저는 PRISM을 **"에세이·합격 예측 전용"** 으로 대체하거나 포기.
- **한 문장:** PRISM이 ChatGPT Plus 대신 선택될 이유는 **"합격 예측 + 에세이 전용 rubric + 학부모 공유"** 이며, 더 나은 점은 **맥락 지속 + 한국어 입시 용어**, 더 못한 점은 **범용성·속도·모델 지능** 이다.

### vs CollegeVine (무료)
- CollegeVine은 미국 기반 무료 대학 매칭 + AI. 한국어 불가 + 한국 국제학교 특수성(예: IB/AP 혼재, TOEFL 필수) 반영 부족.
- PRISM 차별점 **3가지**: (a) 한국어 UI + 한국 대치동 용어 대응, (b) admission_results RAG가 한국 국제학교 합격 사례 기반, (c) 학부모 리포트 (한국 시장 specific).
- **리스크:** CollegeVine이 한국 런칭하면 Free 티어 경쟁력 상실. PRISM은 그 전에 **"한국 사례 데이터 lock-in"** 을 만들어야 함 (합격 사례 수천 건 축적).
- **한 문장:** PRISM이 CollegeVine 대신 선택될 이유는 **한국어 + 한국 국제학교 사례 데이터** 이며, 더 나은 점은 **로컬라이즈드 UX**, 더 못한 점은 **영어권 리소스 깊이·커뮤니티**.

### vs 대치동 컨설팅 (월 50~100만 원)
- PRISM은 **대체품 아님**. 저가 대안도 아님. **보조 도구**.
- **공존 포지셔닝:** "기본 분석·일정·에세이 초안은 PRISM, 전략 결정·최종 첨삭은 컨설턴트". 컨설턴트도 PRISM 리포트를 자료로 쓸 수 있음 (B2B 파트너십 각).
- 문제: 컨설턴트 고객은 "이미 돈 많이 내는 중" → **월 ₩9,900 추가 저항 낮음**. 민준 페르소나 = potential yes.
- **한 문장:** PRISM이 컨설팅 대신 선택될 이유는 **없음(대체 불가)** 이며, 더 나은 점은 **가격·접근성·24/7**, 더 못한 점은 **책임·휴먼 관계·현장감**.

### vs 네이버 카페 + 유튜브 (무료)
- 집단 지성의 강점: **실시간 업데이트 + 익명 디테일**. 약점: **신뢰도 편차 + 정보 노이즈**.
- PRISM의 1인 맞춤 AI가 커뮤니티 대비 "시간 절약"은 명확. 그러나 **"내 상황이 정확히 반영된다는 확신"** 을 유저가 느껴야 함. 지금 PRISM은 그 확신 생성 UX가 약함.
- **한 문장:** PRISM이 네이버 카페 대신 선택될 이유는 **시간 절약 + 개인화** 이며, 더 나은 점은 **일관성·24/7·한 번에 답**, 더 못한 점은 **최신성·현장 사례·공짜**.

## Phase D: 가격 탄력성 분석

| 가격 (Basic / Premium) | Basic 전환율 | Premium 전환율 | 느낌 | 근거 |
|---|---|---|---|---|
| ₩4,900 / ₩9,900 | **5~7%** | **1.5~2%** | 싸다 | "한 달 커피값" 심리 → 무의식 결제. 단, ARPU 낮아 LTV 부담. |
| ₩9,900 / ₩19,900 (현재) | **2~3%** | **0.4~0.8%** | 적정 | 한국 SaaS 구독 평균. 인강 1회차(₩15,000) 근방. |
| ₩14,900 / ₩29,900 | **1~1.5%** | **0.2~0.4%** | 비싸다 | "AI 앱이 인강 1개월치" 저항 발생. |
| ₩19,900 / ₩39,900 | **0.3~0.5%** | **0.05~0.1%** | 많이 비싸다 | "이 돈이면 컨설팅 한 번" 심리. |

**해석:**
- 현재 가격 ₩9,900은 **sweet spot 근처**. 낮춰도 conversion 2~3배 늘지만 ARPU 절반 → 총 매출 비슷. 결정적 차이 없음.
- **Basic → Premium 가격 점프 (2배)** 는 가치 점프가 2배라고 인식 안 됨.
  - 현재 Premium-only: 에세이 첨삭, What-If, 스펙 분석, PDF, 학부모 리포트.
  - 이 중 **실제로 매주 쓸 건 에세이 첨삭 하나**. 나머지는 1회성.
  - → "Premium은 에세이 첨삭 1회=₩2,000 × 10회/월" 프레이밍이 아니면 **2배 가격 정당화 안 됨**.

**연간 할인 매력도:**
- Basic 연 ₩79,000 (월 환산 ₩6,583, 33% 할인). 매력적이지만 **"1년 약정"** 심리 장벽 큼. 대입 결과 나오면 해지할 가능성 → 월간 선호.
- Premium 연 ₩149,000 (월 ₩12,417, 38% 할인). 합격자 부모 1회성 결제로는 매력.

**권고:** 가격은 유지. 대신 **"1주 무료 체험 + 즉시 결제 후 7일 내 취소 가능"** 을 전면에 (이미 pricing 페이지에 있음 — 더 크게 강조).

## Phase H: 출시 후 3개월 시나리오

**가정:** 런칭 D-day 2026-05 초, 런치 3개월 = 2026-07 말. 마케팅 예산 월 ₩1,000,000 (저예산 인플루언서·네이버 카페).

### 시나리오 A (낙관) — 확률 20%
- 가입자: 3,000
- 유료 전환: Basic 75명 (2.5%) + Premium 15명 (0.5%) = 90명
- 월 매출: 75 × ₩9,900 + 15 × ₩19,900 = **₩1,041,000**
- 성공 요인: Phase G의 #1·#2 빠르게 배포, 맘카페·오르비에서 입소문.

### 시나리오 B (현실) — 확률 55%
- 가입자: 1,500
- 유료 전환: Basic 25명 (1.7%) + Premium 4명 (0.3%) = 29명
- 월 매출: 25 × ₩9,900 + 4 × ₩19,900 = **₩327,100**
- 문제: D+7 재방문 15%, Premium 전환 부진, 학부모 결정권 장벽.

### 시나리오 C (비관) — 확률 25%
- 가입자: 600
- 유료 전환: Basic 6명 (1%) + Premium 1명 = 7명
- 월 매출: 6 × ₩9,900 + 1 × ₩19,900 = **₩79,300**
- 실패 원인: "ChatGPT로 되는 거 왜 돈 내?" 지배 여론. 리텐션 붕괴. CAC > LTV 구조 확정.

### 가중 평균 월 매출 (3개월 시점)

```
0.20 × 1,041,000 + 0.55 × 327,100 + 0.25 × 79,300
= 208,200 + 179,905 + 19,825
= ₩407,930
```

**3개월 시점 가중 평균: 월 ₩408,000.**

**확장 6~12개월 외삽:** Phase G의 #1~#3 반영 + 컨텐츠 마케팅 축적 → 6개월 시점 ₩1,500,000~2,400,000 / 월, 12개월 ₩3,000,000~5,000,000 / 월.

**월 1억 매출 도달 필요 조건:**
- Basic ₩9,900 기준 약 10,000명 유료 구독자 필요.
- 유료 전환율 2%로 유지한다면 **50만 MAU** 필요 → 현재 시장(한국 국제학교 재학생 약 2~3만 명 + 해외고 ~5만 명) 규모상 **불가능**.
- 월 1억은 **(a) 객단가 상향(Premium 70%+ 비중, B2B 컨설팅사 번들) + (b) 해외 시장(싱가포르·중국계 국제학교)** 없이는 수학적으로 불가.

**인건비 정당화:**
- 개발자 1인 연봉 기회비용 ≈ ₩80M. 6개월 손익 분기: 월 매출 ₩6~8M 필요.
- **현실 시나리오로는 6개월 내 손익 불가**. 12개월 내 Premium 비중 상향 성공 시 가능.
- → **런치는 OK, 단 "학생 상대 B2C 단독으로 월 1억"은 포기**. 컨설팅사 B2B, 국제학교 단체 라이선스, 해외 확장 전제.

## 결론: 결제 기능 넣기 전에 해야 할 일

### 결제 gate 오픈 **전에** 반드시:

**1. "AI 참조 근거" 위젯 (0.5일)** — Phase G #1. 이거 없으면 "그냥 GPT" 여론을 이길 수 없음. 런치 day 1부터 필요.

**2. 학부모 리포트 샘플 PDF (2일)** — Phase G #2. 학부모 결정권자 체험 자산. 이게 없으면 Premium 전환 0에 수렴.

### 결제 gate 오픈 **후 2주 내** (측정 → 개선):

**3. D+7 재방문율 계측** — Amplitude/PostHog 같은 분석 툴 이미 붙였는지 확인. 안 붙었으면 오픈 전 필수.

**4. 유료 전환 drop-off 지점 추적** — /pricing 노출률, 결제창 도달률, Toss 결제 성공률.

**5. 자동 플래너 기능 (1주)** — 2주 치 데이터 나오면 D+7 재방문이 진짜 15%대면 즉시 개발 투입.

### 결제 gate 즉시 오픈 OK인가?

**Yes, 단 위 #1·#2를 런치 동시에 포함.** 현재 코드 상태는 "결제를 받을 수 있는 기술적 저선"은 통과. "유저가 자발적으로 결제할 만한 심리적 저선"은 #1·#2 두 가지를 추가한 후.

### 솔직한 최종 정리

PRISM은 **"지민 같은 12학년 에세이 시즌 유저"에게는 즉시 payable** 한 제품이다. 그들은 ChatGPT와 분리된 "입시 전용 도구"의 가치를 이해하고, ₩19,900이 컨설턴트 1시간 대비 저렴함을 안다.

문제는 **시장 크기**. 이 페르소나는 한국 국제학교 12학년 중 **상위 10~15%** (약 3~5천 명). 월 ₩19,900 × 1,500명 = 월 ₩3,000만. 여기가 **현실적 천장**.

**월 1억을 원한다면** — 제품을 (a) 10~11학년 일상 사용 도구로 확장 (자동 플래너 + 로드맵), (b) 학부모 자산으로 재설계 (리포트 샘플 + 한국어 내러티브), (c) 컨설팅사 B2B 번들 개시. 이 셋 없이는 런치 자체는 가능하지만 월 1억은 12개월 내 불가.

**출시 권고: Go. 다만 "에세이 시즌 제품"으로 자리 잡을 각오를 하고, 월 1억은 Phase 2 로드맵으로 분리할 것.**
