# 학부모 페르소나 — Stage 3 #13 Phase 1

평가자 원문 (`docs/PRISM_FINAL_DESIGN_AUDIT_2026_04.md` Phase 5 #13):
> 학부모 전용 뷰 (2주). `/parent-report` 확장. 큰 글자, 단순 네비, 요약 중심.
> Elite 결제 전환의 주 축이라면 반드시 별도 디자인 트랙.

본 문서는 학부모 뷰 디자인을 위한 페르소나 + 시나리오 + 디자인 원칙. 코드 변경 없음.

## 1. 페르소나

### P1 · "지원자 어머니" (45–55세)
- **거주**: 서울/경기 일반 학군
- **언어**: 한국어 only. 영어 단어 등장하면 거부감
- **입시 지식**: 한국 정시·수시는 친숙. 미국 입시 용어(Common App, Reach/Target, AP)는 거의 모름
- **디지털 환경**: 카카오톡 + 네이버 + 토스 + 쿠팡. SNS는 카카오스토리/밴드
- **디바이스**: 핸드폰 90% (안드로이드 중심). 데스크톱 거의 안 씀
- **결제 의사결정 권한**: ★★★★★ (실질 결정자, 카드도 본인 명의)
- **사용 빈도 추정**: 주 1회 (학생이 보여주거나 알림 받았을 때)
- **자녀와 함께 보는지**: 첫 진입은 학생이 보여줌, 이후엔 혼자 확인
- **동기**: 자녀 합격, 비용 정당화, 다른 엄마들과 비교
- **두려움**: 비싼 컨설팅(월 30만+), 사기, 시간 낭비, 자녀가 잘못된 방향
- **핵심 질문**:
  1. "내 아이는 잘 하고 있나?"
  2. "이 돈(월 14.9만) 값 하나?"
  3. "친구 아이는 어디 가나?"

### P2 · "유학파 아버지" (45–60세)
- **거주**: 서울 (강남/서초/판교 거주 비율 높음)
- **언어**: 한국어 + 영어 (업무 영어 능숙)
- **입시 지식**: 본인 유학 경험 있음. 단 10–20년 전이라 현재 기준과 괴리
- **디지털 환경**: 평균 이상. 노션/슬랙 사용. Mac 친숙
- **디바이스**: 핸드폰 60% + 데스크톱 40%. 태블릿 보유율 높음
- **결제 의사결정 권한**: ★★★☆☆ (어머니와 협의, 본인이 디지털 도구 평가 담당)
- **사용 빈도 추정**: 월 2–3회 (deep dive 형)
- **자녀와 함께 보는지**: 가끔. 데이터 분석은 혼자, 진로 대화는 함께
- **동기**: 자녀 더 좋은 결과. 본인 유학 경험과 비교/연결
- **핵심 질문**:
  1. "내가 알던 입시와 어떻게 달라졌나?"
  2. "데이터가 신뢰할 만한가?"
  3. "차라리 명문대 동문 컨설팅이 낫지 않나?"

### P3 · "교육 매니저 어머니" (40–50세)
- **거주**: 강남/대치/목동/판교 등 입시 중심지
- **언어**: 한국어 only. 입시 한국어 용어는 매우 능숙
- **입시 지식**: 한국 입시 전문가 수준. 카페/단톡방 정보 활발
- **디지털 환경**: 평균 (네이버 카페·인스타·당근·맘카페 위주)
- **디바이스**: 핸드폰 80%
- **결제 의사결정 권한**: ★★★★★ (주도적, 단톡방 추천 영향 큼)
- **사용 빈도 추정**: 주 2–3회 (모니터링형)
- **자녀와 함께 보는지**: 학생보다 본인이 더 자주 봄
- **동기**: 효율적 시간/돈 분배. 다른 학부모와의 비교 우위
- **두려움**: 오프라인 컨설팅 비용 폭발 (월 50–100만 추세), 단톡방에서 뒤처지는 것
- **핵심 질문**:
  1. "오프라인 컨설팅 대신 이걸로 충분한가?"
  2. "다른 엄마들도 쓰는 건가?"
  3. "데이터를 단톡방에 공유해도 되나?"

## 2. 핵심 시나리오

### 시나리오 A · 결제 전 학부모 보여주기 (마케팅)
- **트리거**: 학생이 "엄마, 이거 봐. AI가 분석해줬어"
- **디바이스**: 학부모가 핸드폰 한 손으로 받아봄 (학생 핸드폰)
- **결정 시점**: 30초~3분 안에 "괜찮네 / 사기 같아" 판단
- **필요한 정보**:
  - 한 화면에 보이는 요약
  - 큰 숫자 (점수/확률)
  - 명확한 신뢰 신호 (출처, 데이터 기준)
- **다음 행동**: 결제 / 일주일 더 두고 본다 / 거절
- **이탈 지점**:
  - 영어 용어 첫 등장 → 0.5초 만에 닫음
  - 작은 글자 → 줌 안 되면 그만 봄
  - 광고 같은 톤 → 즉시 의심

### 시나리오 B · Elite 구독 후 정기 모니터링
- **트리거**: 주간 리포트 알림 (이메일 또는 카카오톡 메시지) — *현재 미구현, Phase 2 검토*
- **디바이스**: 핸드폰. 알림 → 링크 클릭 → 페이지 진입
- **머무는 시간**: 30초–2분
- **필요한 정보**:
  - 이번 주 변화 (+/− 명확)
  - 자녀가 한 일 (에세이 N개, 매칭 N개)
  - 자녀와 통화할 때 격려할 포인트
- **다음 행동**:
  - 자녀에게 카톡 한 줄 ("잘하고 있네")
  - 단톡방 공유
  - PDF 저장 (보관용)
- **이탈 지점**: 데이터 변화 없음 → "왜 돈 내지?"

### 시나리오 C · 결제 의사결정
- **트리거**: 학생이 "Elite 신청해도 돼?" 부탁
- **디바이스**: 핸드폰 (P1/P3) / 데스크톱 (P2)
- **머무는 시간**: 5–15분 (단톡방·검색 병행)
- **필요한 정보**:
  - 가격
  - 오프라인 컨설팅과의 비교
  - 환불 정책
  - 다른 가족의 결과 (사회적 증거)
- **다음 행동**:
  - 카드 결제 (P1/P3)
  - 배우자와 상의 (P2 → 어머니에게 전달)
  - 단톡방 의견 묻기
- **이탈 지점**:
  - 가격이 한 페이지에 정직하게 안 나오면 → 의심
  - "Free trial" 영어 표기 → 거부

## 3. 학부모 뷰 디자인 원칙

### 3-1. 시각
- **최소 글자 크기**: `text-base` (16px). 본문 기본은 `text-lg` (18px)
- **본문 줄 간격**: `leading-relaxed`(1.625) 이상
- **색상 톤**: primary 오렌지 유지하되 채도 낮춤. 카드 배경은 `bg-card` 위주, 그라디언트 최소화
- **장식 최소**: blur, glow, gradient overlay 학부모 페이지에서 절제 ("복잡 = 사기" 인식)
- **폰트**: Pretendard 우선. 영문은 동일 폰트의 라틴 글리프
- **다크모드**: 지원 유지. 50대도 야간 다크모드 사용 증가
- **터치 영역**: 버튼 최소 48×48px (시니어 친화)

### 3-2. 콘텐츠
- **한 화면 한 메시지**: 카드당 한 가지 사실, 두 개 이상 fact 섞지 않음
- **영어 용어 매핑**:
  - `essay` → "에세이"
  - `GPA` → "내신 평점 (GPA)"
  - `SAT` → "SAT 시험" (브랜드 유지)
  - `Common App` → "공통 지원서 (Common App)"
  - `Reach/Target/Safety` → "도전/적정/안전" (한국 입시 어감 차용)
  - `Early App` → "수시 (Early)"
- **숫자 강조**: 백분위, 합격률, 전월 대비 +/−. tabular-nums로 정렬
- **비교 표현**: "작년 합격자 평균보다 +0.3 GPA", "지난주보다 에세이 1개 더"
- **시간 표현**: "이번 주", "다음 달", "1년 전" — ISO 날짜는 보조로만
- **의문형 헤딩 OK**: "우리 아이 이번 주 어땠나요?" — 학부모에게 친근

### 3-3. 네비
- **BottomNav 없음**: 학부모는 한 페이지만 보면 됨
- **상단**: 뒤로가기 + 닫기만
- **메뉴 없음**: 햄버거/사이드바 제거
- **스와이프 제스처 최소**: 시니어는 의도치 않은 제스처 무서워함
- **외부 공유**: 카카오톡 공유 버튼 우선, 그다음 PDF 저장

### 3-4. 인터랙션
- **클릭 = 명확한 텍스트 버튼**: 아이콘만 있는 버튼 금지 (애매함 → 안 누름)
- **로딩 상태**: "불러오는 중..." 큰 글자 (스켈레톤만으로는 의심)
- **에러 메시지**: 친절체 ("문제가 생겼어요. 잠시 후 다시 시도해주세요.")
- **신뢰 신호 노출**: 데이터 기준일, 출처, "AI 자동 생성" 명시
- **결제 페이지 link**: 명시적 텍스트. "지금 결제" 같은 강압 카피 금지 → "Elite 자세히 보기"
